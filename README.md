# BZ Blockcourse 2026 — Spatial Transcriptomics

This repository accompanies the BZ Blockcourse 2026 session on **spatial transcriptomics** and provides a runnable tutorial notebook.

## Contents

- `bz_blockcourse_2026_spatial_tutorial.ipynb` — the hands-on tutorial notebook
- `bz_blockcourse_2026_spatial_tutorial_coding_solutions.ipynb` — the hands-on tutorial notebook with coding solutions
- `bz_blockcourse_2026_spatial_ct.csv` — example cell-type annotations used in the notebook

## Before you start

Please create your **own copy of the Google Colab notebook** before starting the tutorial.

1.Open the notebook link below.
2. Click File → Save a copy in Drive.
3. Work only in your personal copy so that you can edit and run the code.
